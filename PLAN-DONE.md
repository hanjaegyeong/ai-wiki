## 키워드 대규모 확장 검토 (2026-04-04 완료)

결론: 1000개까지 백엔드 없이 정적 파일로 충분. 100개 넘기는 시점에 data.js 분리만 하면 됨.

- 1000개 × 3KB = ~3MB raw, gzip 시 ~1MB (이미지 몇 장 수준)
- JS filter() 1000개 → 1ms 미만, DOM 렌더 수십 ms
- 수정은 Claude Code가 하므로 파일 크기 자체는 관리 문제 없음
- 백엔드 필요 시점: 사용자별 기능(북마크, 인증 등) 추가할 때
