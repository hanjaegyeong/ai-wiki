# /batch-add

여러 키워드를 한 번에 AI Wiki에 추가한다. 카테고리나 주제를 입력하면 관련 키워드를 먼저 선정한 뒤, 각 키워드를 `/add-keyword`와 동일한 방식으로 처리한다.

## 사용법
```
/batch-add 에이전트 관련 키워드 5개
/batch-add "프롬프팅 기법" 추가 키워드들
/batch-add 최신 트렌드 키워드 10개
```

## 워크플로우

### 1단계: 키워드 선정
- 사용자가 지정한 주제/카테고리에 대해 WebSearch로 조사
- 기존 index.html의 `D` 배열을 읽어 이미 있는 키워드 확인
- 중복되지 않는 새 키워드 목록을 선정하여 사용자에게 확인 요청

### 2단계: 승인 후 순차 처리
- 사용자가 키워드 목록을 승인하면 각 키워드에 대해 `/add-keyword`와 동일한 3단계 서브에이전트 처리 실행
- 키워드별로 Agent 3개를 병렬로 띄우되, 키워드 간에는 순차 처리 (index.html 동시 수정 방지)

### 3단계: 다국어 번역
- 각 키워드의 `sum`과 `det`를 영어(en), 중국어(zh), 일본어(ja)로 번역
- `I18N_CONTENT` 객체에 번역 추가

### 4단계: 최종 보고
- 추가된 키워드 수, 카테고리별 분포, 평균 Heat, 번역 완료 여부를 보고
